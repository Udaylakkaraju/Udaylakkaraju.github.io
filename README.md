# udaylakkaraju.me

Source for my personal portfolio site, live at [udaylakkaraju.me](https://udaylakkaraju.me).

Static site (HTML/CSS/JS) showcasing my data analytics projects — supply chain SLA risk, streaming content investment analytics, payment fraud review, and pediatric care-pathway analytics — plus skills, background, and contact info.

`scripts/sync_payment_fraud_metrics.py` pulls current headline metrics from the [Payment-Fraud-Detection](https://github.com/Udaylakkaraju/Payment-Fraud-Detection) repo so the project card stays in sync with that project's latest numbers.

## Stack

Plain HTML, CSS, and JavaScript — no build step. Deployed via GitHub Pages with a custom domain (see `CNAME`).
